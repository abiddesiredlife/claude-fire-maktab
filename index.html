مرحلہ ۲: "Create a project" — نام رکھیں "maktab-fatima"
مرحلہ ۳: "Realtime Database" → "Create Database" → Test mode
مرحلہ ۴: Project Settings → SDK Config → نیچے config کاپی کریں
مرحلہ ۵: نیچے خانے میں config پیسٹ کریں اور "جوڑیں" دبائیں
